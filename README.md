
## 뮤지컬 프로젝트 (Backend)
<hr>

### 목적

Spring boot와 JPA로 RestAPI를 구현하는 경험을 하고자 시작한 학습용 뮤지컬 추천 프로그램입니다.
더 좋은 서비스를 만들기 위한 방법을 고민하며 기능을 확장할 예정입니다.

### 기술 스택
<hr>

+ Java 11
+ Spring 2.7.10
+ Spring Boot
+ Spring Data JPA
+ Gradle

+ MySQL

+ Swagger
+ Postman

### 새로 학습한 기술
<hr>

+ querydsl 페이징
+ Spring Security
+ JWT
